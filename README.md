# 선암사 사운드트랙 (SEONAMSA TEMPLE SOUNDTRACK)

## 프로젝트 개요
여름 무드의 음악 플레이어 랜딩 페이지입니다. 바이닐 레코드 회전 애니메이션과 글래스모피즘 UI를 사용했습니다.

---

## ✅ 완성된 기능

- 🎵 MP3 오디오 재생 / 일시정지
- ⏭ 이전 / 다음 트랙 이동 (앨범 좌우 화살표 + 하단 버튼)
- 📃 트랙리스트 표시 및 클릭 선택
- 🔀 셔플 재생
- 🔁 반복 모드 (없음 / 전체 / 한 곡)
- ⏱ 재생 진행바 (클릭·드래그 탐색 지원)
- 💿 바이닐 레코드 회전 애니메이션
- 🖼 트랙별 앨범 커버 이미지
- ⌨️ 키보드 단축키 (Space: 재생, ←→: 5초 이동, Shift+←→: 트랙 이동)
- 📱 반응형 디자인 (모바일 / 태블릿 / 데스크탑)
- 🌅 배경 이미지 지원 (background.jpg)

---

## 📁 파일 구조

```
index.html              메인 페이지
css/style.css           스타일시트
js/player.js            플레이어 로직
images/
  background.jpg        ← 배경 이미지 (업로드 필요)
  cover01.jpg           ← 트랙 1 앨범 커버 (업로드 필요)
  cover02.jpg           ~ cover05.jpg
audio/
  track01.mp3           ← 트랙 1 음원 (업로드 필요)
  track02.mp3           ~ track05.mp3
```

---

## 🔧 트랙 정보 수정 방법

`js/player.js` 파일 상단의 `TRACKS` 배열을 수정하세요:

```javascript
const TRACKS = [
  {
    id: 1,
    title: '트랙 제목',      // 곡 이름
    artist: '아티스트',      // 아티스트명
    src: 'audio/track01.mp3', // 오디오 파일 경로
    cover: 'images/cover01.jpg', // 앨범 커버 이미지 경로
  },
  // ... 나머지 트랙
];
```

---

## 🖼 배경 / 이미지 업로드 안내

| 파일 경로 | 설명 | 권장 크기 |
|---|---|---|
| `images/background.jpg` | 배경 이미지 | 1920×1080px 이상 |
| `images/cover01.jpg` ~ `cover05.jpg` | 각 트랙 앨범 커버 | 500×500px |

---

## ⌨️ 키보드 단축키

| 키 | 동작 |
|---|---|
| `Space` | 재생 / 일시정지 |
| `←` / `→` | 5초 뒤/앞으로 |
| `Shift + ←` / `→` | 이전 / 다음 트랙 |
